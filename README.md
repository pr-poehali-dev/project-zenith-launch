# Skal Ventures - Инвестиционный лендинг

![Preview](https://cdn.poehali.dev/templates/skal-ventures-ru/preview-ru.jpg)

Премиальный лендинг для венчурного фонда или инвестиционной компании. Включает Hero-секцию с интерактивными 3D-частицами на WebGL (Three.js), минималистичный хедер с навигацией, типографику Sentient и анимации при наведении. Идеален для финтех-стартапов и инвестиционных платформ.
